# ecommerce-project
A simple ecommerce website project for Web Programming subject.   This project showcases a fashion &amp; lifestyle store with product listings, sales dashboard, and a contact form.
## Features
- Responsive navigation bar with dropdown
- Product listing cards with price and add-to-cart button
- Dashboard with sales overview chart
- Contact page with feedback form

## Technology Used
- HTML5
- CSS3
- Bootstrap 5

## Usage
Open `index.html` in Google Chrome browser.  
Navigate through Home, Products, Dashboard, and Contact pages.  
Fill in the contact form to simulate feedback submission.
